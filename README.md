### How to Run

1. Clone the project
2. Install the dependencies:

```bash
npm i
```
Start the server:

```bash
npm run dev
```
Alternatively, you can run the docker:
```bash
docker run -d sepideh1989/rick-and-morty:latest
```
And open this URL in the browser:
```bash
http://127.0.0.1:5173/
```
